# Sessions Plugin for Claude Code

Connect Claude Code to [Sessions](https://sessions.shotgun.dev) - record improvements to your coding profile.

## Installation

```bash
/plugin marketplace add chrishayen/sessions-claude
/plugin install sessions@sessions-claude
```

## Usage

Once installed, the `improve` tool is available. Claude Code will use it to record improvements to your profile based on your coding sessions.

## Authentication

The plugin uses OAuth for authentication. You'll be prompted to sign in to Sessions when first connecting.
